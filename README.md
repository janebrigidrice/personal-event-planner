
# Personal Event Planner

This is a React and Vite web application that helps users organise and manage personal or professional events.

## Features

- User registration
- User login
- Dashboard for upcoming events
- Add new events
- Edit existing events
- Delete events
- Help page
- Responsive design
- Context API for state management
- localStorage for saving user and event data

## Installation

1. Open the project folder in the terminal.
2. Run:
   - `npm install`
   - `npm run dev`

## Technologies Used

- React
- Vite
- JavaScript
- JSX
- React Router
- Context API
- CSS

## Project Structure

- `src/components/`
- `src/context/`
- `src/pages/`
- `src/App.jsx`
- `src/main.jsx`
- `src/index.css`
